Project Info

TheTrustedEngine is a custom Search Engine, built from the ground up, that is focussed on user privacy and safe content filtering.
This is useful if you have to do some research on the web and you are worried about getting fake data or news: TheTrustedEngine, as the name says, does all the work for you, displaying only trusted content, which is personally verified before publishing.
We even have a personal blog coming soon!
